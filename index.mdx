O **Revele JS Plugin SDK** (`js-plugin-lib`) é a biblioteca core baseada em Node.js (TypeScript) desenvolvida para construir integrações dinâmicas para o ecossistema Revele. Ele atua como uma ponte de comunicação bi-direcional (IPC - Inter-Process Communication) entre processos filhos em Node.js e o orquestrador principal em Rust.

## Principais Recursos

- **Abstração de IPC**: Gerencia conexões Socket (Unix Domain Sockets e Windows Named Pipes) automaticamente.
- **RPC Automático**: Permite que o plugin "exponha" funções Javascript para serem chamadas nativamente pelo Rust.
- **Server-Driven UI**: Permite criar interfaces de usuário no Frontend React diretamente pelo Backend Node.js através da classe `UIBuilder`.
- **Mini-ORM (Banco de Dados)**: Acesso direto e seguro aos dados pelo Rust, orquestrado e retornado via Promise.

## Casos de Uso

O SDK é ideal para plugar funcionalidades complexas que requerem integrações com terceiros, automações ou processos persistentes, como:

- Bots do WhatsApp (ex: usando Baileys)
- Integrações de ERPs (ex: Mercos, Bling)
- Crawlers com Puppeteer
